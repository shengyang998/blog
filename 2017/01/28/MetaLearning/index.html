Meta Learning 元学习

Meta 是什么？

MetaLearning 元学习，首先要理解元这个词，Meta。

由于 Meta 的观念翻译上相当困难，所以在翻译上也有许多不同译名，如前所述的metadata，便有人翻译为诠释资料、元资料。其含义是“关于什么的什么”

Meta Learning 是什么？

元学习 即是关于学习的学习，也就是 学习的方法

为什么要进行 Meta Learning

如何进行 Meta Learning

这要求我们在学习的时候更加关注学习方法，关注自己的内心，体验自己是如何进行学习的，这样学习的时候自我感觉如何等等

具体方法论

我总结出来的：

  1. 总览全部知识点

  2. 找出所有感兴趣的生词，记下来
    这里不是狭义的生词，这里泛指所有不能甚解的知识点

  3. 想办法弄懂那些生词，弄懂一个划掉一个

  4. 总结分享
    包括自己解决时候的思路和感受，要记下这些东西需要你一直，或者说大部分时间都要有一种超然的态度：不要局限在问题中，不要局限在生词中。要跳出来，不仅仅跳出问题，为了体验自己的思路和感受，还要跳出自己。总之，难度不小
