Coordinate System

iOS 的坐标系统

每一个 view 都有它的 coordinate system。 原点在（0，0）， 拥有宽度和高度属性。
在 iOS 中使用 bounds 来描述 coordinate system。
iOS 中的 frame 的 origin 属性会含有相对于 superview 的 coordinate system 坐标。
